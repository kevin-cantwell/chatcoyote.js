About chatcoyote.js

chatcoyote.js is a simple way to add instantaneous chat to your website or blog. It's really just a simple plugin that appends an iframe to the dom and opens or creates a Chat Coyote session.

Chat Coyote is a chat client I built using the pretty amazing meteor.js framework, which is still in beta. If you haven't heard of meteor yet, I highly recommend you check it out.

The jQuery plugin can be found/forked here.

In comes in two forms:

  1. A jQuery plugin that allows you to attach a little chat room to any element on your page.
  2. A bookmarklet that fires up an instant chat room based on the website you're viewing at any moment. Anyone group of people use the bookmarklet on the same site will be instantly added to the same chat room.
