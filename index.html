About Chatcoyote

Chatcoyote is a simple way to add real-time chat to your website or blog.

In comes in two forms:

  1. A jQuery plugin that allows you to attach a little chat room to any element on your page.
  2. A bookmarklet that fires up an instant chat room based on the website you're viewing at any moment. Anyone else viewing the same site and using Chatcoyote can thus converse with each other in real-time.
